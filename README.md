## Demo

[Codepen](https://codepen.io/kkern/pen/LKmvjx)

## Quickstart

Only these files are necessary:

```
src/index.html
src/blocks/_modules.scss 
src/js/index.js
```


```bash
git clone git@github.com:regenrek/aspect-ratio-autofill-height-width-lazyload.git
yarn
yarn dev

```
